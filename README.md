# FileNotes App

**FileNotes Pro** is a desktop-based digital notes application built with **Python & CustomTkinter**.  
The app stores notes locally using a file-based system and supports autosave, starred notes, fast search, and PDF export.


---

## Key Features

- Create, read, edit, and delete notes
- Mark important notes as Starred
- Real-time note search
- Automatic autosave while typing
- Import `.txt` files
- Export notes to PDF
- Last updated timestamp
- Sidebar counters (All / Starred / Trash)
- Modern UI with CustomTkinter
